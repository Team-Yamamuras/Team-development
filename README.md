# ながのCAKE
このサイトは、長野県にある小さな洋菓子店「ながのCAKE」の
商品を通販するためにECサイト開発しました。
DMM webcamp 2ヶ月目のチーム制作フェーズで作られたサイトです。
(架空)

# Implement

* user
会員登録・編集・退会
商品の閲覧・カートに入れる・注文
ジャンル検索
注文履歴の確認

* admin
ジャンル登録・編集・非表示選択
商品の登録・編集・削除・非表示選択
注文履歴の確認・進捗ステータスの変更
会員情報の確認・編集
